
These ldif files serve a number of roles:


1) its contents are used in simpletests for drupal ldap modules

2) its contents are inserted in ldap.pixotech.com test ldap server to
which helps user's debug their drupal ldap module installs.

3) they can be used for your own local reference ldap by replacing
"activedirectory,dc=ldap,dc=pixotech,dc=com" with your test ldap's dn.
