
======================================
Provisioning to LDAP from Drupal Notes

Drupal.org pages:
-- http://groups.drupal.org/node/167304
-- http://groups.drupal.org/node/159604

1.  Need to be able to use ldap_user module with ldap authentication and ldap authorization disabled.
2.  LDAP server configuration needs to be able to work without fields that are required for ldap to drupal user provisioning such as puid and mail.


=============================
Special Fields and Attributes

-------------
property.mail
-- if ldap to drupal user provisioning, not configurable in ldap user, configurable in ldap server
-- Otherwise, configurable in ldap user


-------------
property.name
-- if ldap to drupal user provisioning, not configurable in ldap user, configurable in ldap server
-- Otherwise, configurable in ldap user

------------
field.ldap_user_puid_sid
field.ldap_user_puid
field.ldap_user_puid_property
field.ldap_user_current_dn

-- only used in ldap to drupal user provisioning, not configurable
-- only visible in ldap user mapping UI if ldap to drupal user provisioning enabled
